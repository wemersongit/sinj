Parametros:
partial_salvar_json
	Salva no disco o json das tabelas secundárias
full_salvar_json
	Salva no disco o json todas as tabelas
normas_e_diarios_salvar_json
	Salva no disco o json das tabelas normas e diarios
normas_e_diarios_json
	Envia ao LBRest o json das tabelas normas e diarios salvas no disco
full_json
	Envia ao LBRest o json de todas tabelas salvas no disco
full_banco
	Envia ao LBRest o json de todas tabelas do banco
parcial_banco
	Envia ao LBRest o json des tabelas secundárias do banco
parcial_json
	Envia ao LBRest o json des tabelas salvas no disco
normas_json
	Envia ao LBRest o json da tabela de normas salva no disco
normas_banco
	Envia ao LBRest o json da tabela de normas do banco
diarios_json
	Envia ao LBRest o json da tabela de diario salva no disco
diarios_banco
	Envia ao LBRest o json da tabela de diario do banco
arquivos
	Atualiza no LBRest o json das tabelas de normas e diarios com arquivos
Sem nenhum parametro:
	Envia ao LBRest o json das tabelas de normas e diarios com arquivos do banco
